# Open FPGA Robot Lab

## Plataforma Educacional de FPGA + Robótica + IA Embarcada

Open FPGA Robot Lab é um projeto educacional aberto focado em:

- Aprendizagem de FPGA
- Sistemas digitais
- Robótica
- Sistemas embarcados
- Aceleração por hardware
- Integração FPGA + ESP32
- Experimentos práticos e visuais

A primeira plataforma utilizada neste repositório será a:

## Terasic DE0-Nano FPGA Development Kit

Características:

- FPGA Intel/Altera Cyclone IV
- USB-Blaster onboard
- GPIOs de expansão
- Acelerômetro
- ADC
- SDRAM
- LEDs e switches

# Projetos Planejados

## 01 - Blink LED
Seu primeiro projeto FPGA.

## 02 - Contador Binário
Aprendendo registradores e temporização.

## 03 - Knight Rider LEDs
Animação visual usando FSM.

## 04 - Debounce de Botão
Aprendendo filtragem digital.

## 05 - Gerador PWM
Controle de LEDs e motores.

## 06 - Comunicação UART
FPGA conversando com ESP32.

## 07 - Leitura do Acelerômetro
Usando o ADXL345 onboard.

## 08 - FPGA + ESP32 Bridge
Arquitetura híbrida embarcada.
