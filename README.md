## Educational FPGA + Robotics + Embedded AI Platform

Open FPGA Robot Lab is a didactic and open educational project focused on:

- FPGA learning
- Digital systems
- Robotics
- Embedded systems
- Hardware acceleration
- FPGA + ESP32 integration
- Practical and visual experiments

The first platform used in this repository is the:

## Terasic DE0-Nano FPGA Development Kit

Featuring:

- Intel/Altera Cyclone IV FPGA
- USB-Blaster onboard
- GPIO expansion
- Accelerometer
- ADC
- SDRAM
- LEDs and switches
