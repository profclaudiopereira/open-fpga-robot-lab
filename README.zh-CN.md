## FPGA + 机器人 + 嵌入式 AI 教育平台

Open FPGA Robot Lab 是一个开放式教育项目，重点包括：

- FPGA 学习
- 数字系统
- 机器人
- 嵌入式系统
- 硬件加速
- FPGA 与 ESP32 集成
- 实践实验

本项目使用的第一块开发板：

## Terasic DE0-Nano FPGA 开发板

主要特性：

- Intel/Altera Cyclone IV FPGA
- 板载 USB-Blaster
- GPIO 扩展
- 加速度传感器
- ADC
- SDRAM
- LED 与拨码开关

# 计划中的项目

## 01 - Blink LED
第一个 FPGA 项目。

## 02 - Binary Counter
学习寄存器与时序。

## 03 - Knight Rider LEDs
使用 FSM 实现 LED 动画。

## 04 - Button Debounce
学习数字滤波。

## 05 - PWM Generator
控制 LED 与电机。

## 06 - UART Communication
FPGA 与 ESP32 通信。

## 07 - Accelerometer Reader
使用板载 ADXL345。

## 08 - FPGA + ESP32 Bridge
混合嵌入式架构。
